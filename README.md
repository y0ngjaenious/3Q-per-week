# 3Q-per-week

## Description
YBIGTA 2021 코딩테스트 스터디입니다. 6개월간 최소 일주일에 3문제씩 풀어 각종 기업 코딩테스트에 대비합니다.

### Rule
- 일주일에 최소 3문제 풀고 개인 레포에 업로드(플랫폼 무관, 언어 무관), 정리 형식은 무관
- 좋은 문제는 공유 레포에 공유
- 3문제를 풀지 못하였을 경우 페널티 5000원/문제
- 페널티 7만원 이상은 퇴출
